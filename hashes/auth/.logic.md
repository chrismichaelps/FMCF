# AuthService: Logic Blueprint

**Shard:** `@root/packages/auth/src/AuthService.ts`
**State:** `BigInt:0x1`
**Fidelity:** Signature

---

## Sign-Up Algorithm

1. **Decode** — `Schema.decodeUnknown(LoginPayload)(rawBody)` → `Effect<LoginPayload, ValidationError>`
2. **Check Duplicate** — `yield* UserRepo.findByEmail(payload.email)` → if `Option.isSome(result)`, fail with `DuplicateEmailError`
3. **Hash** — `Effect.tryPromise({ try: () => argon2.hash(payload.password), catch: (e) => new CryptoError({ cause: e }) })` → `HashedPassword`
4. **Persist** — `yield* UserRepo.create({ email, passwordHash })` → `User`
5. **Sign Token** — `yield* TokenService.sign({ userId: user.id })` → `string`
6. **Encode** — `Schema.encode(AuthResponse)({ id, email, token, createdAt })` → wire response

## Login Algorithm

1. **Decode** — `Schema.decodeUnknown(LoginPayload)(rawBody)` → `Effect<LoginPayload, ValidationError>`
2. **Fetch** — `yield* UserRepo.findByEmail(payload.email)` → `Option<User>`. If `Option.isNone`, fail `InvalidCredentialsError`
3. **Verify** — `Effect.tryPromise({ try: () => argon2.verify(user.passwordHash, payload.password), catch: ... })` → `boolean`. If `false`, fail `InvalidCredentialsError`
4. **Sign Token** — `yield* TokenService.sign({ userId: user.id })` → `string`
5. **Encode** — `Schema.encode(AuthResponse)(...)` → wire response

## Negative Logic (Prohibited Paths)

- ❌ Do NOT store raw passwords in any form.
- ❌ Do NOT catch errors silently — every failure goes through the `E` channel.
- ❌ Do NOT return `user.passwordHash` in any response.
- ❌ Do NOT use `bcrypt` — use `argon2` only (as per MVP spec).
- ❌ Do NOT access the DB directly — always go through `UserRepo` Tag.
